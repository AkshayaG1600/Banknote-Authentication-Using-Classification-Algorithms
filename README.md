**💰 Banknote Classification using Machine Learning**

This project focuses on building a machine learning classification system to determine whether a banknote is authentic or fake based on statistical features extracted from images of banknotes. Multiple classification algorithms are implemented and compared to identify the most accurate model.
Detecting counterfeit banknotes is a critical issue in financial systems. Manual detection is:

•	Time-consuming

•	Prone to human error

•	Not scalable

**This project solves the problem by:**

•	Automating the detection process using machine learning

•	Providing a reliable and scalable solution for classification of banknotes

**🔍 Workflow Breakdown**

Import Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, XGBoost

Dataset: https://www.kaggle.com/datasets/shantanuss/banknote-authentication-uci

Model Training:

  •	Decision Tree Classifier

  •	Random Forest Classifier

  •	XGBoost Classifier

  •	Logistic Regression

Model Evaluation:

  •	Accuracy comparison

  •	Confusion Matrix visualization

  •	Classification Report

  •	ROC Curve

The project compares multiple models and identifies the best-performing algorithm based on accuracy and evaluation metrics. Typically, ensemble methods like Random Forest and XGBoost perform better due to their ability to handle complex patterns.
